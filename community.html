Community Resources

Reporting problems or suggesting features

Submit a new issue in the github Issues tracker for the appropriate gladiusjs repository. If you're not sure which repository contains the code you're concerned with, the core Issues tracker is a reasonable choice.

Chat

The #games IRC channel on irc.mozilla.org.

Mailing List

For more persistent discussion with better reach, subscribe to and use the community-games mailing list.
